    COVID-19 HUB

    A website that would act as a hub for doctors and nurses at hospitals around Australia to view information and perform COVID-19 related tasks. Users were able to view bed supply figures and projections, check patients in and out, view custom built COVID-19 frequency maps of Australia, and manage notifications and tasks. The website had permission level based access to certain features through a login portal.

  • Featured Project

    Block World

    A Java project which utlises classes and files themed around a blockworld. The GUI displays the game world and has options for users to interact with the world by performing actions on the world using control elements in the game window. The block world is made up of a variety of different block types and a character (builder) who can manipulate the blocks. The user can also load and save world states.

  • Featured Project

    Snake for a ATmega324A Controller

    Created the game Snake for the AVR ATmega324A microcontroller. It receives input from a number of sources and outputs information to an LED matrix.

    This version of Snake implements simple movement around the display and food consumption. It also keeps track of your score, has special types of food for bonus points, has an increasing speed of play, sound effects, and the use of controller buttons. The code interacts with the serial port and the LED display and is run using Atmel Studio Environment.

    • C
    • AVR ATmega324A
    • Atmel Studio
